# shein-extractor

تطبيق سطح مكتب لاستخراج منتجات روابط سلة SHEIN وعرضها وحفظ سجل التحليلات.

## التشغيل

```powershell
.\.venv\Scripts\Activate.ps1
python app.py
```

## تصدير PDF

- يصبح زر `تصدير البيانات` متاحًا بعد اكتمال تحميل جميع صور المنتجات.
- يصدّر التقرير جميع منتجات التحليل، حتى عند تفعيل بحث يخفي بعض صفوف الجدول.
- يتضمن التقرير بيانات العميل والطلب والرابط والإحصائيات وجدول المنتجات والصور.
- يقتصر اسم المنتج داخل التقرير على أول 100 حرف.
- يُقترح حفظ التقارير داخل `exports/`، مع إمكانية اختيار مكان آخر.

إذا تعذر تحميل صورة واحدة أو أكثر، يبقى زر التصدير معطّلًا. أعد فتح التحليل من السجل بعد التحقق من الاتصال لإعادة محاولة تحميل الصور.
