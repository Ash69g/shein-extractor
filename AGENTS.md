# تعليمات توثيق المشروع

- حدّث `PROJECT_LOG.md` بعد كل تغيير جوهري في الكود، أو قرار تقني، أو تحليل، أو تجربة، أو نتيجة اختبار.
- اكتب التوثيق بالعربية الواضحة، مع إبقاء أسماء الملفات والأوامر والحقول البرمجية بلغتها الأصلية.
- أضف الأحداث الجديدة تحت قسم السجل الزمني، وحدّث أقسام الحالة الحالية والخطوات التالية عند الحاجة.
- سجّل الأوامر المهمة ونتائجها الفعلية، ولا تسجّل نتيجة لم يتم التحقق منها.
- لا تحذف التاريخ السابق؛ صحّح المعلومة القديمة بإضافة توضيح مؤرخ إذا تغيّر الاستنتاج.
